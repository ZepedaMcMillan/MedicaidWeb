Lists

To apply Bullets and Numbering, simply select the text that you wish to format as a list and then select one of the pre-defined list styles from the toolbar Styles drop-down list (see Styles and Formatting for details).

To view or modify Bullets and Numbering, in your document, select the list item(s) and right-click to select Bullets and Numbering.

Lists defined using 'Bullets and Numbering' cannot be used in content that spans multiple fragments. To apply continuous numbering across fragments, Business Content Studio's Insert Numbering method must be used.
