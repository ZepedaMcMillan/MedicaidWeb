Document Comparison

When editing or regenerating a document (see Data Entry View), it may be necessary to apply to your current version edits made either during your current session (making a comparison with the 'old' version), or in another version previously stored to file. To this end, the Review Case Editor allows the side-by-side comparison of two instances of a Review Case. Each view graphically illustrates the differences in rules and content between the two versions of the document. In addition, you can copy changes from one version to the other.

To open a Document Comparison:

  • From a regenerated document, click the Comparison icon to open the comparison frame directly
  • From any document, including a regenerated document, select File/Compare to view the Compare Review Case window. Here you can browse to select the Review Case version with which you wish to compare the open version.

The comparison frame is displayed in the current window. Each view of this frame is described below.

If there are no differences between the selected document templates, then a message to this effect is displayed.

In the terminology employed here:

  • The Target is the Review Case from which you opened the comparison frame
  • The Source is the Review Case with which you have elected to compare the Target.

Toolbar

The comparison frame offers three views for comparison of the Source and Target (see View Final and View Source/Target and View Source/Target/Final). Each view has the same toolbar options:

Icon

Option

Function

-

Apply

Apply any changes as Accepted or Rejected, as in the current Final view of the template

-

Cancel

Quit the Comparison utility without applying any changes to the template

Previous

Shift focus to the change last in focus

Next

Shift focus to the change next in focus

Accept Change

Accept the change currently in focus. This accepts the change from the Source identified in the Target

Reject Change

Reject the change currently in focus. When applied, this updates the Target with content from the Source

Accept All

Accept all of the changes from the Source identified in the Target

Reject All

Reject all of the changes from the Source identified in the Target

View Final

A view of the template in its current and - once all changes have been either Accepted or Rejected - final state

View Source/ Target

A side-by-side comparison of the Source and Target templates

View Source/ Target/ Final

A side-by-side comparison of the Source and Target above a view of the Final.

Toolbar Options

Customize the toolbar by adding and removing buttons.

It is a known limitation that you cannot customize the Document Comparison toolbar when deploying Document Editor Control in Internet Explorer.

* Until changes are applied, their effects are apparent only in the Final view (see View Final).
