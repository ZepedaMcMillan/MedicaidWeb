Exceptions

An Exception is a user-defined error message that has been inserted into a Document Template to identify an issue with an item of data in the Transaction. Where an Exception is encountered by the Thunderhead Engine, the document is re-routed to an external Workflow system. The user can then use the Document Editor to view the document for further editing, approval, and/or review.

A Review Case can include multiple Primary documents.

Until Thunderhead 3.5, a Review Case was typically a document containing an Exception. From Thunderhead 3.5 onwards, other documents can also be considered as Review Cases, or specifically as Portable Review Cases.
