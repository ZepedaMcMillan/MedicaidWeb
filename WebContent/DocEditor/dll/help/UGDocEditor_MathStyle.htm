Math Style Properties

The Style Properties window is accessed via the Add Equation window's Equation Builder.

Here you can define any of the following properties (MathML style attributes) for your equation.

The following table describes the various MathML style attributes and their valid values:

Name

MathML Attribute

Values

Font Style

mathvariant

NOT SET (default) | normal | bold | italic | bold-italic | double-struck | fraktur | bold-fraktur | script | bold-script | sans-serif | bold-sans-serif | sans-serif-italic | sans-serif-bold-italic | monospace

Size

mathsize

small | normal | big | number | %

Foreground Color

mathcolor

#rgb | #rrggbb | html-color-name

Background Color

mathbackground

#rgb | #rrggbb | html-color-name

Display Style

displaystyle

Automatic | True | False

Script Level

scriptlevel

Automatic | 0 | 1 | 2 | +1 | +2 | -1 | -2
