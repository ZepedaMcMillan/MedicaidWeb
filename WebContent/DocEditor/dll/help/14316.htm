Further Documentation

Each Thunderhead installation comes with a full set of supporting documentation, which can be found in your Thunderhead Platform docs folder.

The documents that you will find here include, among many others:

  • Thunderhead Help - Provides help for all screens in each of the Thunderhead components
  • TH Config - Security - Documents User Login, authentication and authorization; WSS Security and Encryption; system accounts; the Thunderhead Database.

A Documentation Map PDF (also to be found in the Docs directory) summarizes the full Thunderhead documentation set.
