Additional Comparison Types

You can identify table insertions, deletions and formatting changes, as well as changes to table rows, columns and cells. Please refer to Tables.

Inserting and deleting a list, and modifying the list items and list order, is also tracked and highlighted. Please refer to Bulleted Lists and Numbered Lists.

When you modify an inline object, the utility treats it as a deletion and insertion. Please refer to Inline Objects.

You can apply formatting changes to a number of objects as well as text. Please refer to Formatting Objects.
