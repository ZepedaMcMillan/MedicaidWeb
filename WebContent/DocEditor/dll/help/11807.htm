Overlay Forms

Overlay Forms are made available for editing within the Document Editor, subject to the usual Edit Rights.

Any fields that you have the right to edit have blue borders, while those that you cannot edit have red borders.

The Fields will be edited in-place on the Form image. The Data Fields will employ data type validation as well as drop-down lists for enumerations. The formatting that is applied is derived from the formatting specified for the data item in the Document Template.

To edit text fields simply by overtyping the current value. To edit fields that contain values derived from Data Items, simply double-click the field to display the appropriate Edit Data Item window (see Edit a Data Item Value).
