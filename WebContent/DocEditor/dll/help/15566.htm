Search all PDFs

You can search for a word or phrase across the full Thunderhead documentation set. To do this:

  1. Open Adobe Reader®
  2. Select Edit | Search
  3. Within the Where would you like to search? section, select Browse for Location from the drop-down
  4. Navigate to your Thunderhead Platform docs folder and click OK
  5. Enter the word or phrase for which you want to search and click Search.

The Results section shows all PDF files where the search term was found. Expand the list to show the search results in context. Click on a search result to open the PDF file at the location of the text shown.
