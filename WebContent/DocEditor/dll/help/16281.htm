Formatting Objects

The Compare feature identifies formatting changes applied to the following objects:

  • Individual piece of alphanumeric text or phrase
  • Paragraph
  • Bulleted list
  • Numbered list
  • Tables and the columns/rows of a table
  • Equations
  • Shared content/Dynamic Content Links and Dynamic URL links
  • Input form controls
  • TOCs
  • Data items
  • Scripts
  • Fragments
