Content Editor

Much of your editing will be done in the Content Editor.

To display the Content Editor, select a Data Item in the tree view and then either:

  1. Click the Edit icon
  2. Press F2.

When you edit a Review Case, the document that you are editing is Channel-specific. However, on re-submission to Thunderhead the Review Case produces output in each Channel specified by the Batch Config.

XML Data Channels are not supported in Review Cases.
